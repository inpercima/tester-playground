# Test angular apollo nestjs graphql - backend

## Getting started

```bash
# all commands used in ./backend
cd backend

# install tools and backend dependencies
yarn
```

## Usage

```bash
# build and starts a server, rebuild after changes, reachable on http://localhost:3000/
yarn start:dev

```

### Tests

```bash
# test
yarn test

# e2e
yarn test:e2e
```
